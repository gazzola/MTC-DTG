O conceito de determinante está ligado ao de matriz, embora seja completamente distinto: enquanto matriz é o conjunto de elementos conforme já mencionado, determinante é o resultado de uma operação aritmética com todos os elementos de uma matriz, que obedece a uma determinada regra. Só se aplica a matrizes quadradas. O prefixo "det" é colocado antes da matriz para indicar determinante.
